# 个人知识库 · 总索引

> **形式：** Markdown 文档（`.md`），在 Cursor 里编辑最顺手；成长中心网页是**目录入口**，不是在线编辑器。  
> **入口：** [职业成长中心](https://ada481131-alt.github.io/ada-portfolio/career.html) → **个人知识库**

---

## 怎么增删改？

| 操作 | 你怎么做 |
|------|----------|
| **新增一篇** | 在 `career/evidence/` 或 `career/evidence/kb/` 新建 `.md`（可复制 [`kb/_template.md`](kb/_template.md)） |
| **改内容** | Cursor 打开该文件直接改；面经类文末有「复盘区」 |
| **删除** | 删掉该 `.md` 文件；若成长中心有卡片，再说一声让我改 `career.html` 或你自己删对应链接 |
| **上成长中心首页** | 在 `career.html`「个人知识库」区块加一张卡片（链到你的新 `.md` 或 `.html`） |
| **网页版** | 重要篇可再做 `.html`（如 Smartico memo）；日常笔记 **md 就够** |

**不会自动同步：** 新建 md 后，网页不会自己出现链接——要么加进本索引，要么加 `career.html` 卡片。

对我说 **`知识库加一篇：xxx`** 我可以帮你建文件 + 改索引 + 改成长中心。

## 文档放哪、怎么命名

| 类型 | 建议路径 / 文件名 | 例子 |
|------|-------------------|------|
| **面经 · 口播 QA** | `evidence/interview-qa-*.md` | `interview-qa-cs-handoff.md` |
| **面经 · 公司真题** | `evidence/kb/interview-公司-岗位-日期.md` | 面试后新建 |
| **项目 · STAR** | `evidence/star-stories.md` 或 `kb/star-*.md` | Story A/B/C |
| **课程笔记** | `evidence/kb/course-课程名.md` | 训练营、专栏 |
| **知识 · 概念** | `evidence/kb/learn-主题.md` | 阈值、RAG、评测 |
| **竞品 / Memo** | `comp-memo-*.md` + 可选 `.html` | Smartico |
| **作业证据** | `metrics-draft`、`jd-benchmark` 等 | Sprint 交付 |

## 面经复盘（通用模板）

每场面完后，在当篇文档末尾写：

1. 公司 / 岗位 / 日期  
2. 问了什么  
3. 答好的 / 答砸的  
4. 下一句要改的  

或对 Cursor 说 **`面试复盘`** / **`知识库加一篇`**.
